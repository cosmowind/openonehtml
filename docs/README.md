# OpenOneHTML GitHub Pages

这个文件夹包含用于GitHub Pages的静态网站文件。

## 结构说明

- `index.html` - 主页入口
- `static-pages/` - 主要的项目页面
- `.nojekyll` - 防止GitHub Pages使用Jekyll处理

## 部署状态

如果部署遇到问题，请检查：

1. 仓库是否为公开状态
2. GitHub Pages设置是否正确指向docs文件夹
3. 等待构建完成（可能需要几分钟）

## 访问地址

部署完成后可以通过以下地址访问：
`https://cosmowind.github.io/openonehtml/`
